# Spec Summary (Lite)

Give service records a screen and `DataAnomaly` its first read path, so the two workbook defects still
unprovable in the running app can be entered, flagged and seen: the stale MOT (the dashboard reads "Not set"
because nothing can create the pass record its countdown derives from) and the 83,000 mi row (which lives on the
Service History sheet and has nowhere to be typed).

The two ship together because neither proves the thesis alone — one supplies the record that trips the detector,
the other is where the flag surfaces. Entering the 83,000 raises exactly one anomaly, the odometer does not
move, the row is kept, and the queue records what was decided about it.
